Link PostgreSQL to this web service

Quiz submissions are stored in Postgres. The variable must be on the same service that runs this Node app (not only on the database component).

Render (typical)

  1. Dashboard → your PostgreSQL instance → use Connect / link to your Web Service.
  2. On the Web Service → Environment: confirm DATABASE_URL (or DATABASE_PRIVATE_URL) appears and is non-empty.
  3. Manual copy: paste the Internal Database URL into the web service as DATABASE_URL if auto-link did not run.
  4. Redeploy or restart the web service, then reload this page.

Railway (typical)

  1. Add a PostgreSQL plugin; open the web service → Variables → reference DATABASE_URL from the Postgres service.
  2. Redeploy, then reload this page.
